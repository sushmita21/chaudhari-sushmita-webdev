Application Developer

At IBM India, I worked as an application developer for a web application that catered to a telecommunication giant's data warehouse. Being the youngest developer in the team, I learnt passionately about the architecture of the application, development, production environment, testing and QA process and the deployment model. The agile model on which the project was based helped me learn the benefits of agile development.
